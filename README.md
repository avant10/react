

# Create a progressive web application with react

##preinstall Node.js snd npm manager

##1
Download the file or git clone https://github.com/avant10/react

## Step 2
cd react

## Step 3
```
yarn install
```
## Step 4


#### Production Mode

In the directory, run:

```
yarn build
```

```
node server.js
```

#### Development Mode

In the directory, run:

```
yarn start-dev
```
